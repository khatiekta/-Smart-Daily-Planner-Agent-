# Smart Daily Planner - Concierge AI Agent

## Track: Concierge Agents

**Description:**  
Smart Daily Planner is an AI-powered concierge agent designed to automate task management. It allows users to:
- Add tasks with natural language
- Set reminders for specific dates and times
- Receive desktop notifications
- Query pending tasks

**Key Concepts Demonstrated:**
1. Task Extraction & Prioritization
2. Reminders & Notifications
3. Command Understanding

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run agent: `python src/main.py`
4. Example commands:
   - "Remind me tomorrow at 4 PM for online class"
   - "What are my tasks?"
